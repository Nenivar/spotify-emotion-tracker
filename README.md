# spotify-emotion-tracker

## Installing

1. `pipenv install`
1. `npm install`
1. create `auth.json`:
    ```
    {
        "username": "",
        "client_id": "",
        "client_secret": "",
        "redirect_uri": "http://127.0.0.1:5000/",
        "scope": "user-read-private"
    }
    ```

## Building

`npm run build`


## Running

`make server`
